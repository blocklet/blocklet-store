# Blocklet Store

## What is blocklet?

**Blocklets** are reusable building blocks to help developers and community users to build things on [ArcBlock](https://www.arcblock.io) platform. A blocklet serves one and only one purpose, reusability is the core design philosophy of blocklets. Reusability makes it easy to compose multiple blocklets to build a more complex application.

Blocklets comes in many forms, including but not limited to:

- A starter template, from which a developer can bootstrap an dapp already connected to ocap-powered blockchain within minutes
- A full-featured DApp, with which a miner or community user can install and get it up and running quickly
- A smart contract, which defines the on-chain pipeline of a business logic and can be deployed to ocap-powered chains with one click

## What is blocklet store?

To help developers find usable blocklets more easily, a Blocklet Store instance can list blocklets built by ArcBlock and community developers. Users can search and view listed blocklets and launch a blocklet with one click.

## How to use blocklet store?

Open your Blocklet Store instance, view the blocklets list, and click the "Launch" button to launch a blocklet.

## Publish to a store

Sign up on your target Blocklet Store instance and apply for a developer NFT. Then build your own blocklet and publish it to that store. Learn more about how to build blocklets in the ArcBlock documentation.
